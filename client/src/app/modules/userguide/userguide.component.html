User Guide

When using the pencil, you can change the line width with Line Width bar

When using the pencil brush, you can change the line width with Line Width bar

When using the pen, you can change the line maximal and minimal width with their respective bars

When using the pipette tool, left click to extract the color of any element and store it as your primary color

You can also right click to extract the color of any element and store it as your secondary color

When using the rectangle tool, you can change the line width with Line Width bar

You can also choose between 3 options to set the contour and the fill

You may want to draw a square? Press shift while using this tool

When using the ellipse tool, you can change the line width with Line Width bar

You can also choose between 3 options to set the contour and the fill

You may want to draw a circle? Press shift while using this tool

Start line: click, Add new segment: click, Finish line: double-click

When using the line tool, you can change the line width with Line Width bar

You can choose between a full line or a dashed line by selecting the type

You can choose between 3 options to set the syle of the line junctions

For the point-style junctions you have to set the diameter

By pressing backspace, you remove the curent line segment

By pressing escape, you delete the current line

You may want to close your the path? Press shift while finishing a line

When using the feather tool, you can change the line width with the Line Width bar

You can choose the feather angle with the Rotation angle bar

You can also choose the feather angle with a mouse scroll to increment the angle by 15 degrees

By pressing alt and mouse scorlling, you can increment the angle by 1 degree

When using the spray tool, you can change the spray diameter with the Line Width bar

You can choose the number of pixels you color at a time with the Pixels number bar

When using the stamp tool, you can change the stamp size with the size bar

You can choose the stamp angle with the rotation angle bar

You can also choose the stamp angle with a mouse scroll to increment the angle by 15 degrees

By pressing alt and mouse scorlling, you can increment the angle by 1 degree

You can choose between 5 Options to set the desired stamp tu use

When using the text tool, you can press anywhere on the drawing zone to create a text element

You can change the font with the Font Family selection field

You can choose the font size with the Font Size input field

You can choose whether bold or italc text or both

You can also choose between 3 Options to set the text alignement

When using the polygon tool, you can change the line width with the Line Width bar

You can choose the polygon's number of faces with the Faces input field

You can also choose between 3 Options to set the contour and the fill

When using the selection tool, you have to left click and expand the selection zone to select the disired elements

You can use the clip board on the selected elements to copy, cut, past, duplicate and delete

You can also select all the items of the drawing zone with the Select All button

When using the bucket tool, you can left click on any closed area to fill it with the selected primary color

You can also set a contour and it's width with the Line Width bar

When using the eraser tool, you can change the eraser size with the Size bar

While left clicking, you can drag the eraser over the elements to erase

When using the color applicator tool, set the primary color of an element with a left click on it

You can also set the secondary color of an element with a right click on it
